TechStore — веб-приложение интернет-магазина**

Учебный проект интернет-магазина электронной техники. Приложение состоит из фронтенда и бэкенда, использует Supabase как удалённую базу данных и реализовано на стеке PERN (PostgreSQL, Express, React, Node.js). Проект представляет собой минимально жизнеспособную версию (MVP), включающую основные функции электронной коммерции.

---

## **Функциональность**

Приложение поддерживает:

* просмотр каталога товаров;
* карточку товара;
* поиск и сортировку;
* фильтрацию по цене и бренду;
* корзину с возможностью изменения количества;
* оформление заказа;
* регистрацию, авторизацию и изменение профиля пользователя.

---

## **Технологии**

**Frontend:** React, React Router, Axios, Tailwind CSS, Vite
**Backend:** Node.js, Express.js
**Database:** Supabase (PostgreSQL)

---

# **Как запустить проект локально**

## 1. Клонировать репозиторий

```bash
git clone https://github.com/your-repo.git
cd TechStore
```

---

## 2. Запуск backend

```bash
cd backend
npm install
```

Создайте файл **.env**:

```
SUPABASE_URL=ваш_supabase_url
SUPABASE_KEY=ваш_supabase_anon_key
PORT=4000
```

Запустите сервер:

```bash
npm start
```

Бэкенд будет работать по адресу:
**[http://localhost:4000](http://localhost:4000)**

---

## 3. Запуск frontend

```bash
cd ../frontend
npm install
```

Создайте файл **.env**:

```
VITE_API_BASE_URL=http://localhost:4000
```

Запуск:

```bash
npm run dev
```

Фронтенд откроется по адресу:
**[http://localhost:5173](http://localhost:5173)**

# **Примечания**

* Перед запуском фронтенда обязательно должен работать backend.
* Supabase используется как облачная база, поэтому локальная настройка PostgreSQL не требуется.
* Проект создан в учебных целях, но структура позволяет в дальнейшем расширить функциональность.
